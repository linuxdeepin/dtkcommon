## dtkcommon

A public project for building DTK Library

## Dependencies

### build on debian/uos

requirements:

- pkg-config
- libglib2.0-bin
- qttools5-dev-tools
- qtbase5-private-dev

### Build from source code

```bash
cmake -B build && cmake --build build
```
