## 项目名称

dtkcommon：一个用来构建DTK库的公共项目

## 编译依赖

### 在debian/uos上构建

软件包依赖:

- pkg-config
- libglib2.0-bin
- qttools5-dev-tools
- qtbase5-private-dev

### 从源代码构建

```bash
mkdir build && cd build && qmake ../ && make
```
